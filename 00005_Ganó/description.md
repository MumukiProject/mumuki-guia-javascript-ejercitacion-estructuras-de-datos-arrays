> Definí una función `gano` que tome como argumento un array `tragamonedas` con 5 símbolos y devuelva `true` si son iguales y `false` sino. Si el array tiene más de 5 símbolos, sólo debe comparar los 5 primeros.
>
```javascript
ム gano(['⭐ '⭐️', '⭐️', '💫', '✨']) // 
false
ム gano(['💫', '💫', '💫', '💫', '💫']) 
true
ム gano(['💫', '💫', '💫', '💫', '💫', '⭐️']) 
true
```
